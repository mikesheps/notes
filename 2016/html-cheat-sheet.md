# HTML Cheat sheet



### Paragraph
`<p>whatup!</p>`

### Unordered list
`<ul></ul>`

### list Item
`<li></li>`








### lists

```html
<ul><-- Unordered list-->
  <li>one</li><-- List Item-->
  <li>two</li><-- List Item-->
  <li>three</li><-- List Item-->
</ul>
```



### Basic html page
```html
<!DOCTYPE html>
<html>
	<head>
		<title>First font size change</title>
	</head>
	<body>
		<p> Some text for you to make tiny! </p>
	</body>
</html>
```
